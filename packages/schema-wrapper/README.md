# Mosaic Python API

## Generate the API from the JSON schama

`uv run generate.py ../../docs/public/schema/v0.10.0.json`.

## Run Tests

`uv run pytest -v`

## Run the ruff formatter

`uv run ruff format` and `uv run ruff check`.

## Test notebook

`uv run jupyter lab` and then open `example.ipynb`.
